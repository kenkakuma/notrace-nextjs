   - 检查"任务清单"中标记为"🔄 进行中"的任务
   - 阅读该任务的"遇到的问题"和"下一步"

2. **恢复Git分支**
   ```bash
   git checkout feature/website-redesign
   git status
   ```

3. **运行开发服务器**
   ```bash
   npm run dev
   ```

4. **打开浏览器测试**
   - 访问 http://localhost:3000
   - 测试当前任务的变更

5. **继续编码**
   - 按照当前任务的"子任务"列表继续
